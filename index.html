Logistic Regression Binary Predictions

A logistic regression machine learning model can be used in cases where the outcome of a prediction can be classified in a binary fashion (e.g. Yes/No, Dog/Cat, Has/Doesn't Have). In our analysis, where we are predicting whether an individual has an income above or below a certain threshold (i.e. $50K per year), a logistic regression model is a good candidate for prediction since we are only classifying into two groups.

The same feature set was used across the different machine learning models tested, including the logistic regrression model. Based on the input features included, each individual was then classified as having an income above or below $50K per year. The criteria included in our feature set considered type of employment (Workclass), Education level, Marital Status, specific Occupation, current Relationship, Race, Gender, and hours worked per week.

In comparison to other machine learning models tested, the logistic regression model scored the highest in terms of balanced accuracy score (0.7534), which was slightly better than the Support Vector Machine (0.7478) or Random Forest model (0.74). Given this provided the highest level of accuracy, this model was used to develop our interactive income predictor built into our web page. Users can enter in their own set of criteria, based on the features included, and see if they are likely to have an income above or below $50K.

Logistic regression is one of many options availble in determining an appropriate machine learning model to build out predictions. As shown in this project, the scores were very similar among other potential models. This demonstrates the need for testing multiple model options prior to deciding which is the best fit to predict outcomes based on your input data set.
