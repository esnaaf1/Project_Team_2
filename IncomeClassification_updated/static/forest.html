Random Forest Classification

Our dataset consisted of 30162 rows of data for our machine learning models, with 10 input features: age, hours-per-week, education, marital-status, work-class, occupation, gender, relationship, race, native country..

Using Scikit-learn's train_test_split module to split 75% of features (X) and target (y) data into training data and 25% for testing data.

Set n_estimators=100,random_state=1, we have a balanced accuracy score of 0.74 and precision of 0.81.
